# 🍆 가지가지 하네! (Gaji-Gaji-Hane)

흙 속에서 튀어나오는 기상천외한 가지들을 사정없이 터치해서 뽑아내는 병맛 가득한 하이퍼 캐주얼 터치 액션 게임입니다.

[게임하러가기](https://luminen529.github.io/gaji-gaji-hane/)

---

## 🎮 게임 방법

- **일반 가지** (🍆): 기본 가지입니다. 터치해서 수확하면 10점을 획득합니다.
- **헬창 가지** (💪): 근육질 가지입니다. 몸이 단단해서 3번 연타해야 수확되며 30점을 획득합니다.
- **황금 가지** (✨): 전설의 영물입니다. 매우 빠르게 사라지지만 수확하면 100점을 획득합니다.
- **가짜 고구마** (🍠): 가지밭에 침입한 방해꾼입니다. 터치하면 20점이 감점되고 콤보가 리셋됩니다.
- **피버 타임**: 콤보를 계속 이어가 게이지를 채우면 광란의 피버 타임이 발동합니다. 피버 타임 동안에는 배경이 화려하게 변하며 7초간 수확 점수가 2배가 됩니다.
